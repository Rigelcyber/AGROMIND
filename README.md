# 🌱 AgroMind — Smart Farming AI Platform

A browser-based smart farming dashboard with **real AI plant disease detection** powered by your own MobileNetV2 model (`AgroMind.keras`), live weather, sensor simulation, and device control.

---

## 🚀 Quick Start (3 steps)

### Step 1 — Convert the model (run once)
```bash
pip install tensorflowjs tensorflow
python convert_model.py
```
This reads `public/model/AgroMind.keras` and outputs `public/model/tfjs_model/`.

### Step 2 — Start the app
```bash
npm install
npm start
```
**OR** in VS Code: right-click `public/index.html` → **Open with Live Server**

### Step 3 — Use it
1. Click **Sign In**
2. Go to **Disease Detection** in the sidebar
3. Drop a leaf photo → click **Analyse Disease**
4. Get instant AI results — runs 100% in your browser 🎉

## 🧠 How the AI Works

| What | Detail |
|---|---|
| Architecture | MobileNetV2 (1.0, 160×160) |
| Classes | 39 PlantVillage crop disease classes |
| Input | 160×160 RGB image, normalised ÷127.5 − 1 |
| Output | 39 sigmoid probabilities |
| Runtime | TensorFlow.js 4.11.0, 100% in-browser |
| Model file | `AgroMind.keras` (Keras 3.13.2) |

The model auto-loads from `model/tfjs_model/model.json` when the page opens.
If the converted model isn't found, the app runs in **demo mode** automatically.
